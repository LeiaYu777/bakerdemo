# FAQ

## 1. 页面空白或打不开怎么办？

请先执行：

```bash
npm install
npm run dev
```

并确认访问地址为 `http://localhost:5173`。

## 2. 这个 Demo 是否接入了真实摄像头和模型？

没有。当前版本仅用于业务演示，视觉结果和字幕均为假数据或脚本生成。

## 3. 可否替换成客户自己的门店数据？

可以。建议替换 `src/data/mockData.js` 中 `stores` 生成逻辑，保持字段结构不变。

## 4. 为什么不同页面数据会变化？

项目使用伪随机生成逻辑。当前已使用固定种子，保证同一次构建内数据稳定可复现。

## 5. 如何关闭自动播放？

进入 `客户演示模式`，点击 `暂停` 按钮；或将“播放结束行为”设置为“播放一次后停止”。

## 6. 如何更改字幕语言默认值？

修改 `src/pages/DemoModePage.jsx` 中 `subtitleLang` 的初始 state。

## 7. 如何新增演示脚本页面？

在 `createDemoScript(store)` 中新增一个脚本对象，并在 `renderCurrentStep()` 中补对应渲染逻辑。

## 8. 构建体积过大警告是否影响演示？

通常不影响功能演示。若需优化，可考虑按页面做动态加载（code splitting）。
