# 客户演示操作手册

## 1. 目标

在 5-8 分钟内，向客户完整展示：
- 总部如何看全局
- 单店如何发现问题
- 场景如何给出整改建议
- 系统如何自动讲解（演示模式 + 字幕）

## 2. 推荐讲解顺序

1. 首页 Dashboard（约 90 秒）
2. 门店详情（约 120 秒）
3. 8 个视觉场景（约 180 秒）
4. 巡检表单（约 60 秒）
5. 客户演示模式自动播放（约 60 秒）

## 3. 现场演示脚本（可直接口播）

## 3.1 Dashboard

- “这里是总部视角，16 家门店状态一屏可见。”
- “我们关注四个核心指标：评分、整改、产品、环境。”
- “右下角老板评估面板把财务、运营、品牌、风险统一到一个决策视图。”

## 3.2 门店详情

- “进入单店后，可以看门店评分与异常数量。”
- “中间是 8 个重点视觉巡检场景，每个场景都能定位问题位置和整改建议。”
- “点击异常可查看整改状态和复核图片，形成闭环。”

## 3.3 巡检表单

- “门店人员可在表单里打分并上传照片。”
- “系统自动给出视觉分析结果，辅助质检和复盘。”

## 3.4 自动演示模式

- “现在切到自动模式，系统按脚本逐页讲解。”
- “可以调节每页停留时长，支持中文字幕、英文字幕、日文字幕。”
- “用于展会、路演和无人值守大屏轮播都很合适。”

## 4. 演示模式控制说明

- `播放/暂停`：控制自动切页
- `上一页/下一页`：人工控场
- `跳转脚本页`：直接跳到指定讲解片段
- `自动切页时长`：30-60 秒
- `播放结束行为`：循环/停止
- `字幕开关`：开启/关闭
- `字幕语言`：中文/英文/日语
- `全屏`：适配大屏演示

## 5. 高风险问题应答模板

- Q：这是实时接入摄像头吗？
- A：当前 Demo 为离线假数据演示，真实项目可对接现场摄像头或巡检终端。

- Q：识别准确率是多少？
- A：本 Demo 不展示真实模型精度，只展示业务流程和落地形态。

- Q：可否接入你们现有系统？
- A：可，通常通过 API/消息总线对接门店、工单、BI、权限体系。
