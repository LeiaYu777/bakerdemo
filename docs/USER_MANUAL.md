# 用户手册

## 1. 产品定位

本系统是用于客户展示的烘焙店智能巡检 Demo，覆盖巡检总览、门店详情、视觉场景识别、巡检表单和自动演示模式。

## 2. 页面功能说明

## 2.1 首页 Dashboard

入口：左侧菜单 `首页 Dashboard`

可查看：
- 核心指标：平均巡检评分、整改完成率、产品合格率、环境达标率
- 评分趋势图（近 10 天）
- 异常数量分布图（16 家门店）
- 门店概览卡片（点击可跳转门店详情）
- 老板评估指标面板（点击行可展开详细说明）

## 2.2 门店列表

入口：左侧菜单 `门店列表`

可查看：
- 门店名称、地址、巡检评分、异常数量
- 点击任意行或“查看详情”进入门店详情页

## 2.3 门店详情

入口：左侧菜单 `门店详情` 或从其他页跳转

可查看：
- 门店基础信息与核心指标
- 四维度评分：产品、环境、员工、设备
- 8 个重点视觉巡检场景卡片
- 异常列表（点击异常查看整改状态和虚拟照片）

8 个视觉场景：
1. 缺货 / 空位检测
2. 商品错位 / 串货
3. 排面整齐度 / 对齐度检测
4. 价签合规检测
5. 商品朝向 / 正反检测
6. 堆头 / 促销区合规检测
7. 货架清洁 & 异物检测
8. 重点 / 新品陈列合规检测

## 2.4 巡检表单

入口：左侧菜单 `巡检表单`

可操作：
- 对环境、原料、设备、产品、服务进行 1-5 分评分
- 点击“上传照片（模拟）”生成占位图
- 自动生成视觉分析结果（合格/异常）和建议文本

## 2.5 客户演示模式

入口：左侧菜单 `客户演示模式`

可操作：
- 自动切页播放（30-60 秒可调）
- 暂停/继续、上一页/下一页、跳转任意脚本页
- 选择循环播放或播放一次后停止
- 字幕开关 + 字幕语言切换（中文/英文/日语）
- 全屏展示

## 3. 状态颜色说明

- 绿色：达标 / 合格 / 已完成
- 橙色：关注 / 整改中
- 红色：异常 / 预警 / 待整改

## 4. 常见使用流程（客户演示建议）

1. 从 `首页 Dashboard` 讲全局价值
2. 进入 `门店详情` 讲业务闭环与视觉场景
3. 打开 `巡检表单` 讲一线巡检操作
4. 切换 `客户演示模式` 自动播放完整故事线

## 5. 数据说明

- 所有数据均为假数据（随机生成 + 固定样例）
- 无后端依赖
- 无真实 AI 接口调用
